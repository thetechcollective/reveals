How It Works

Create a Markdown file in a GitHub repository, then use our loader to turn it into a presentation:

# Welcome to My Presentation A brief introduction to my topic. <!--slide--> ## Main Content - Point one - Point two - Point three <!--slide--> ## Conclusion Thank you for your attention!

Separate slides with <!--slide--> comments and you're ready to present!
